### Getting Started

#### Prerequisites

* heroku
  ```sh
  npm install heroku -g
  ```
* github cli
   ```sh
  brew install gh
  ```
  (for additional info [installation-guide](https://github.com/cli/cli#installation))


#### Installation

1. Clone the repo
   ```sh
   gh repo clone Finnko/frontend-project-lvl4
   ```
2. Install NPM packages
   ```sh
   make install
   ```
3. Start development
   ```sh
   make start
   ```
4. Build
   ```sh
   make build
   ```
